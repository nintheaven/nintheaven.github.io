链表介绍

数组和链表是最基本的数据结构，是实现其他数据结构的基础。。

链表本质上是串在一起的一个个节点。

链表的每个节点必须有一个表示其内容的变量，一般命名为 key 或 val；

链表的每个节点至少要有1或2个表示其指向的变量：prev 和 next，prev指向指向它的前驱元素，next指向它的后继元素；

链表的每个节点中可能还有其它辅助数据，称为卫星数据。

链表的头结点一般称为 head，尾节点称为 tail

链表包括单向链表、双向链表、循环链表。

单向链表只有指向后继元素的 next 指针，双向链表 next 和 prev 都有，循环链表在双向链表的基础上，将 head.prev 指向 tail，并将 tail.next 指向 head。

构造链表

构造单向链表使用如下的数据结构

public class SinglyNode<E> {
    public E val;
    public SinglyNode<E> next;

    public SinglyNode(E val) {
        this.val = val;
    }
}

构造双向链表和循环链表使用如下的数据结构

public class DoublyNode<E> {
    public E val;
    public SinglyNode<E> prev;
    public SinglyNode<E> next;

    public DoublyNode(E val) {
        this.val = val;
    }
}
