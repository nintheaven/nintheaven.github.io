Java实现数据结构-栈Stack

: 数据结构与算法
发布于 : 2022-10-11

栈介绍

栈（Stack）是一种动态集合。

栈采用后进先出（last-in, first-out, LIFO）策略；

栈的添加元素操作称为 push，删除元素操作称为 pop。

对一个空栈执行 pop 操作，称为栈下溢 underflow，反之，当执行 push 操作时，超出了栈的最大容量时，称为栈上溢 overflow。

一个基本的栈数据结构至少要包含以下几个操作：

  • isEmpty()：栈是否为空
  • size：返回栈中元素的数量
  • push(E)：入栈，添加元素到栈顶
  • pop()：出栈，并返回栈顶元素

定义Stack 接口如下，Stack 接口继承 JDK内置的Iterable 接口，要求实现类必须提供重写的 iterator() 方法（迭代器模式）

package adt.stack;

/*栈接口*/
public interface Stack<E> extends Iterable<E> {
    boolean isEmpty();
    int size();
    void push(E item);
    E pop();
}

数组实现

使用 Object[] 存储栈中的元素，使用一个 int 类型的变量 top 表示栈顶，当 top = 0 时，表示栈为空；当 top = 数组长度 时，表示栈已满。

push 时：

  • 先判断是否达到最大容量上限，如果是，抛出 overflow 异常
  • 再判断栈是否已满，如果是，扩容
  • 将元素添加到数组末尾，top 加1

pop 时：

  • 判断栈是否为空，如果是，抛出 underflow 异常
  • top 减1
  • 获取数组中top位置的元素，并将该位置元素置为null
  • 返回数组top位置的元素

扩容：这里只是简单的每次加10个容量，并确保容量不超过上限

时间复杂度

入栈、出栈操作都为 O(1)。

package adt.stack;

import java.util.Arrays;
import java.util.Iterator;

/*栈：数组实现*/
public class ArrayStack<E> implements Stack<E>{
    // 默认容量
    private static final int DEFAULT_CAPACITY = 10;
    // 最大容量
    private static final int MAX_CAPACITY = 100;

    // 存放元素的数组
    private Object[] elements;
    // 栈顶位置
    private int top = 0;

    public ArrayStack() {
        this.elements = new Object[DEFAULT_CAPACITY];
    }

    public ArrayStack(int initialCapacity){
        assert initialCapacity >= 0 && initialCapacity <= MAX_CAPACITY;
        this.elements = new Object[initialCapacity];
    }

    @Override
    public boolean isEmpty() {
        return top == 0;
    }

    @Override
    public int size() {
        return top;
    }

    @Override
    public void push(E item) {
        if (top == MAX_CAPACITY) {
            throw new RuntimeException("overflow");
        }
        if (top+1 > elements.length) {
            grow();
        }
        this.elements[top++] = item;
    }

    @Override
    public E pop() {
        if (top == 0) {
            throw new RuntimeException("underflow");
        }
        E item = (E) elements[--top];
        elements[top] = null;
        return item;
    }

    /*扩容*/
    private void grow(){
        int oldCapacity = elements.length;
        int newCapacity = Math.min(oldCapacity + 10, MAX_CAPACITY);
        elements = Arrays.copyOf(elements,newCapacity);
    }

    @Override
    public Iterator<E> iterator() {
        return new ArrayStackIterator();
    }

    private class ArrayStackIterator implements Iterator<E>{
        int cursor = 0;

        @Override
        public boolean hasNext() {
            return cursor < size();
        }

        @Override
        public E next() {
            E e = (E) elements[cursor++];
            return e;
        }
    }
}

链表实现

使用双向链表实现栈数据结构。只涉及到链表的插入、删除及其迭代。

在栈中维护一个叫 top 的节点，表示栈顶。push和 pop时，只需要操作该节点即可。由于是链表数据结构，所以容量无上限（取决于JVM内存）。

时间复杂度

入栈、出栈操作都为 O(1)。

package adt.stack;

import java.util.Iterator;

/*栈：队列实现*/
public class LinkedStack<E> implements Stack<E> {
    // 头结点
    private ListNode<E> head = null;
    // 栈顶节点
    private ListNode<E> top = null;
    private int size = 0;

    public LinkedStack() {

    }

    @Override
    public boolean isEmpty() {
        return head == null;
    }

    @Override
    public int size() {
        return size;
    }

    @Override
    public void push(E val) {
        if (head == null) {
            head = new ListNode<>(val, null, null);
            top = head;
        } else {
            top.next = new ListNode<>(val, top, null);
            top = top.next;
        }
        size++;
    }

    @Override
    public E pop() {
        if (top == null) {
            throw new RuntimeException("underflow");
        }
        E val = top.val;
        ListNode<E> prev = top.prev;
        if (prev == null) { // 当top为首元素时
            head = null;
            top = null;
        }else {
            prev.next = null;
            top.prev = null;
            top = prev;
        }
        size--;
        return val;
    }

    private class ListNode<E> {
        E val;
        ListNode<E> prev;
        ListNode<E> next;

        public ListNode(E val, ListNode<E> prev, ListNode<E> next) {
            this.val = val;
            this.prev = prev;
            this.next = next;
        }
    }

    @Override
    public Iterator<E> iterator() {
        return new LinkedStackIterator();
    }

    private class LinkedStackIterator implements Iterator<E> {
        private ListNode<E> head = LinkedStack.this.head;

        @Override
        public boolean hasNext() {
            return head != null;
        }

        @Override
        public E next() {
            E val = head.val;
            head = head.next;
            return val;
        }
    }
}

测试

ArrayStack 和 LinkedStack 均实现 Stack 接口，拥有一样的API，使用方法一样

public static void main(String[] args) {
    //Stack<Integer> stack = new ArrayStack<>();
    Stack<Integer> stack = new LinkedStack<>();
    for (int i = 0; i < 10; i++) {
        stack.push(i);
    }

    System.out.println("size = " +stack.size());

    Iterator<Integer> it = stack.iterator();
    while (it.hasNext()){
        System.out.printf("%d ",it.next());
    }
    System.out.println();

    for (int i = 0; i < 10; i++) {
        Integer num = stack.pop();
        System.out.printf("%d ",num);
    }
    System.out.println();
    System.out.println(stack.isEmpty());
}
