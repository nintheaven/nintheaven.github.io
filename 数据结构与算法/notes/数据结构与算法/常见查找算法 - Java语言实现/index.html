常见查找算法 - Java语言实现

2022-10-11

线性查找

public static int linearSearch(int[] nums,int target)
{
    for (int i = 0; i < nums.length; i++) {
        if (nums[i]==target)
        {
            return i;
        }
    }
    return -1;
}

二分查找

public static int binarySearch(int[] nums, int target) {
    int low = 0, high = nums.length;
    while (low < high) {
        int mid = (high - low) / 2 + low;
        if (target < nums[mid]) {
            high = mid-1;
        } else if (target > nums[mid]) {
            low = mid+1;
        } else {
            return mid;
        }
    }
    return -1;
}
