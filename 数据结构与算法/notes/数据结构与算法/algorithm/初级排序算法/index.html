初级排序算法

2022-10-11

选择排序

算法描述

每轮循环中选择最小元素

遍历每个元素，分别与其余元素进行比较，找出未排序部分中的最小元素，将其与未排序部分的首元素交换位置。

复杂度分析

时间复杂度: O(n^2)

空间复杂度: O(1)

算法实现

public static void sort(int[] nums)
{
    int N=nums.length;
    for(int i=0;i<N;i++)
    {
        int min=i;
        for(int j=i+1;j<N;j++)
        {
            // 更新最小值索引
            if(nums[j]<nums[min])
            {
                min=j;
            }
        }
        
        // 交换
        int temp=nums[min];
        nums[min]=nums[i];
        nums[i]=temp;
    }
}

冒泡排序

算法描述

遍历元素，两两比较，并交换位置，将较大的往后移。

每轮循环可以找到剩余元素中的最大的元素。

复杂度分析

时间复杂度: O(n^2)

空间复杂度: O(1)

算法实现

public static void sort(int[] nums)
{
    int N=nums.length;
    for(int i=0;i<N;i++)
    {
        for(int j=0;j<N-i-1;j++)
        {
            if(nums[j]>nums[j+1])
            {
                // 交换位置
                int temp=nums[j];
                nums[j]=nums[j+1];
                nums[j+1]=temp;
            }
        }
    }
}

插入排序

算法描述

遍历元素，将每一个元素插入到其它已经有序的元素中的适当位置。

复杂度分析

时间复杂度: O(n^2)

空间复杂度: O(1)

算法实现

public static  void sort(int[] nums)
{
    int N=nums.length;
    for(int i=1;i<N;i++)
    {
        for(int j=i;j>0;j--)
        {
            if(nums[j]<nums[j-1])
            {
                int temp=nums[j];
                nums[j]=nums[j-1];
                nums[j-1]=temp;
            }
        }
    }
}
