单链表

public class ListNode
{
    public int val;
    public ListNode next;

    public ListNode(int x)
    {
        val = x;
        next = null;
    }
}

迭代链表

public void nodeVals(ListNode head)
{
    ListNode temp=head;
    while(temp!=null)
    {
        System.out.println(temp.val);
        temp=temp.next;
    }
}

使用快慢指针

获取列表的中间节点

public ListNode midNode(ListNode head)
{
    if(head == null)
    {
        return head;
    }
    ListNode fast = head, slow = head; // 快慢指针
    while(fast.next != null && fast.next.next != null)
    {
        fast = fast.next.next;
        slow = slow.next;
    }

    if(fast.next == null) // 奇数个元素
    {
        return slow;
    }
    else // 偶数个元素
    {
        return slow.next;
    }
}

反转链表

public ListNode reverseList(ListNode head)
{
    ListNode pre=null;
    ListNode cur=head;
    ListNode next=null;
    while(cur!=null)
    {
        next=cur.next; // 保存原链表的下一个节点
        cur.next=pre; // 当前节点的下一个指向前一个节点
        pre=cur; // 更新前一个节点为当前节点
        cur=next; // 更新当前节点为原链表的下一个节点
    }
    return pre;
}

合并链表

1->2->3 和 4->5

合并为

1->4->2->5->3

public void mergeTwoList(ListNode head1,ListNode head2)
{
    if(head1==null || head2==null)
    {
        return;
    }

    while(head2!=null)
    {
        ListNode temp1=head1.next;
        ListNode temp2=head2.next;
        head1.next=head2;
        head2.next=temp1;

        head1=temp1;
        head2=temp2;
    }
}
