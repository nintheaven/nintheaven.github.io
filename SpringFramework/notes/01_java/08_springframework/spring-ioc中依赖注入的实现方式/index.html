spring-ioc中依赖注入的实现方式

SpringFramework 2022-10-11

概述

实现DI有以下几种方式

  • 基于构造器
  • 基于Setter方法
  • 基于静态工厂方法

在Spring中，实现配置的方式有3种

  • 基于XML配置的方式
  • 基于注解配置的方式
  • 基于Java代码配置的方式

这里主要以基于XML的方式来说明依赖注入。
