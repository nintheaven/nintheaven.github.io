Windows10+Ubuntu20.04LTS双系统安装

笔记 : Windows
发布于 : 2022-10-11

1. 在Win10中创建空白磁盘

磁盘分区格式：UEFI新式bios+单硬盘

磁盘管理，压缩卷。

如果只能压缩所选磁盘的一部分，说明本地磁盘启用了【系统保护】功能。

此电脑，属性，系统保护，选择要压缩的磁盘分区，配置，选择【禁用系统保护】，确定。

2. 创建Ubuntu系统启动盘

使用rufus，软碟通UltraISO，或者Ventoy等工具，创建U盘启动盘

3. 开始安装

3.1 设置boot启动

根据电脑品牌，启动时，在logo界面按快捷键进入boot设置界面，设置U盘启动

3.2 安装Ubuntu

安装类型：
选择【其它类型】，选择安装位置的时候，选择空闲分区（第一步创建的空白磁盘）。

ubuntu分区：

  • efi，启动引导分区，200M
  • /swap，交换分区，8G
  • /，根分区，50G
  • /home，家目录分区，剩余的全部空间

重要：【安装启动引导器的设备】，选择windows UEFI启动引导分区（Windows Boot Manager），其作用是将windows启动引导目录挂载到ubuntu的efi分区。
