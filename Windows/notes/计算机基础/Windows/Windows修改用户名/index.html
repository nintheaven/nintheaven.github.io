Windows修改用户名

2022-10-11

修改用户名

右键此电脑》》管理》》本地用户和组》》用户，找到要修改的用户，重命名

修改用户home目录名

1.激活管理员账号

右键此电脑》》管理》》本地用户和组》》用户，找到Administrator用户，右键选择属性，取消勾选账户已经用，确认

2.重命名home文件夹名为新的用户名

注销当前账号，用Adminstrator账号登陆。

打开 C:\Users 目录，重命名home文件夹名为新的用户名。

3.修改注册表

在CMD命令行界面，输入 regedit 命令，打开注册表编辑窗口，依次选择

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\ProfileList

ProfileList目录下，每个配置项对应一个Windows系统中的用户。

其中，每个配置项中 ProfileImagePath 变量的值就是该用户的home目录路径。

修改 ProfileImagePath 的值为新的路径即可。

4.注销Administrator用户，登录普通用户。

OK！最好再将Adminstrator用户取消激活。
