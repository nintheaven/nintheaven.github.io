Windows CMD

笔记 : Windows
发布于 : 2022-10-11

查看端口占用情况

netstat -aon 列出所有端口的使用情况

netstat -aon | findstr "8761" 查看包含字符串”8761”的记录行，例如8761端口被占用，使用该命令可以查看使用该端口的进程ID

C:\WINDOWS\system32>netstat -aon | findstr "8761"
  TCP    192.168.2.15:8761      113.96.237.69:443      CLOSE_WAIT      18324

查看指定进程ID的应用名

tasklist | findstr "18324"

C:\WINDOWS\system32>tasklist | findstr "18324"
QQ.exe                       18324 Console                    4    144,160 K

删除顽固文件

DEL /F /A /Q \\?\%1

RD /S /Q \\?\%1

使用以上2个命令制作bat工具，删除任意文件 xxx.bat "文件路径"

磁盘修复

解决 错误0x80070570: 文件或目录损坏且无法读取

chkdsk d: /f

启动后台运行程序

start /b java -jar mms-api.jar
