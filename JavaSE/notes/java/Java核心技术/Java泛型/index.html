Java泛型

: JavaSE
发布于 : 2022-11-30

泛型类

定义泛型类

public class Point<T>
{
    private T x;
    private T y;

    public Point(T x,T y)
    {
        this.x = x;
        this.y = y;
    }

    public T getX()
    {
        return x;
    }
    
    public T getY()
    {
        return y;
    }
}

使用泛型类

Point<Integer> point=new Point<>(1,1);

可以声明多个泛型类型，用逗号分割

public class Point<T,U>{}

泛型方法

定义泛型方法

public class Math
{
    public static <T> T middle(T... nums)
    {
        return nums[(nums.length)/2];
    }
}

使用泛型方法

int mid=Math.<Integer>middle(1,2,3,4);
int mid=Math.middle(1,2,3,4); // 可以省略泛型

泛型约束(类型限定)

可以使用关键字extends限定泛型必须继承某个类，或者实现某些接口。

public static <T extends Comparable> T max(T... nums)
{
    T max=nums[0];
    for(T num: nums)
    {
        if(num.compareTo(max)>0)
        {
            max=num;
        }
    }
    return max;
}

可以根据需要拥有多个接口类型限定，但最多只能有一个类类型限定，而且，它必须是限定列表中的第一个。

限定类型之间用与&分割。

public static <T extends Number & Comparable & Serializable> T max(T... nums)
{
    // ...
}

泛型通配符

使用泛型时，泛型参数除了使用确定的类型外，还可以使用通配符?

public boolean onTopOf(Point<?> point)
{
    return (int)this.y>(int)point.y;
}

通配符除了可以使用extends进行限定外，还可以使用super进行超类型限定

限定泛型类型必须实现Comparable接口

Point<? extends Comparable>

限定泛型类型必须是Integer的超类型(或接口)

Point<? super Integer>

类型擦除

泛型是编译期现象，经过编译后，泛型会被擦除。运行期中，只有原始类型，这就是类型擦除。

  • 如果泛型没有类型限定，其原始类型就是Object
  • 如果只有一个类型限定，其原始类型就是这个限定类型
  • 如果有多个类型限定，其原始类型就是限定列表中的第一个类型

例如

public class Point<T>
{
    private T x;
    private T y;
}

类型擦除后，变为

public class Point
{
    private Object x;
    private Object y;
