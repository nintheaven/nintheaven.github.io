Java中的package,jar,module

: JavaSE
发布于 : 2022-10-11

Java的模块化级别

类

通过访问修饰符设定对象的可见性

包(package)

略

jar(Java Archive)

创建jar包

jar cvf xxx.jar <files>

创建可执行的jar包

  1. 创建一个文件manifest，在其中输入以下内容

    Main-Class: 入口函数类全名

  2. 运行命令

    jar cmf manifest xxx.jar <files>

    当当前项目依赖其它jar包时，需要将其它jar包解压，然后一起归档为同一个jar包。

  3. 启动程序

    java -jar xxx.jar

模块(module)

jdk9新特性

模块描述符: 依赖-导出

必须与Java源码在同一级目录下

module-info.java

module <module-name>
{
    requires <module-names>;
    exports <package-names>;
}

编译，打包模块

javac module-info.java <java_files> -d target

jar cvfe module_name.jar <class_files> -C target

使用模块

java --module-path module_name.jar --module 要运行的模块/要运行的Main类
