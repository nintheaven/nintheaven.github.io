Java开发环境搭建

: JavaSE
发布于 : 2021-05-01

Linux

下载安装

  1. 下载
  2. 安装(解压到 opt/ 目录)

配置环境变量

编辑 ~/.bashrc 文件，添加以下内容

JAVA_HOME="/opt/jdk-15.0.1"
PATH=$PATH:$JAVA_HOME/bin
# 可选
PATH=$PATH:$JAVA_HOME/jre/bin
# 可选
CLASS_PATH=$JAVA_HOME/lib:$JAVA_HOME/lib/tools.jar
export PATH

使其生效

source ~/.bashrc

验证安装

javac --verison

java --version

替换默认的openjdk版本为自己安装的jdk版本

运行命令

sudo update-alternatives --install /usr/bin/java java /opt/jdk-15.0.1/bin/java 300

sudo update-alternatives --install  /usr/bin/javac javac /opt/jdk-15.0.1/bin/javac 300

然后运行命令

sudo update-alternatives --config java

按提示进行配置即可
