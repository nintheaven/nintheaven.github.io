Java脑筋急转弯

2022-10-11

final修饰的基本类型不能被修改，那被修饰的引用类型一定能被修改吗？

答案是不一定

final String s="abc";

private修饰的成员一定不能在其所在类之外被访问吗？

不一定,内部类中的私有成员，对外部类是可见的

public class Test
{
    public static void main(String[] args)
    {
        A a=new A();
        System.out.println(a.value);
    }
    
    static class A {
        private int value=0;
    }
}

被static final修饰变量，一定不能被修改吗？

一不定，本地方法可以随便修改，例如 System.out 这个PrintStream对象。
