Java中的this与super关键字

JavaSE 2022-10-11

this的2个作用

  1. this代表隐式参数的引用
  2. this调用当前类的其它构造器

super的2个作用

  1. super调用超类中的方法
  2. super调用超类中的构造器

使用this与super调用构造器的语句，必须是构造器中的第一条语句，而且只能使用一次，当然，也就不能同时使用它们。

this与super有本质上的不同。
this是当前对象的引用，可以赋值给其它变量，或者作为参数传递，还可以作为返回值，它与普通对象一样。
super仅仅是一个指示编译器调用超类方法的关键字，而不是一个引用。
