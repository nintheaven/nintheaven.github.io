springboot实现定时任务

SpringBoot 2022-10-11

开启定时任务功能

首先，在SpringBoot配置类中，使用注解 @EnableScheduling 开启定时任务功能。

定义定时任务

使用注解的方式

最方便的方式是，在定时任务方法上，使用注解 @Scheduled，即可开启一个定时任务。

这个注解支持不同的属性，用于定义不同行为的定时任务。

默认情况下，定时任务是阻塞执行的。如果同时开启多个定时任务，最好将其定义为异步任务。
