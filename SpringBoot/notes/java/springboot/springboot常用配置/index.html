springboot常用配置

笔记 : SpringBoot
发布于 : 2022-10-11

application.yml 示例

server:
  address: localhost
  port: 8080

debug: false

logging:
  level:
    root: WARN
    org.springframework.security: DEBUG
  file:
    path: /var/logs/
    name: spring-boot.log
  config: classpath:logging-config.xml

spring:
  main:
    banner-mode: console

  datasource:
    url: jdbc:mysql://localhost:3306/test2
    driver-class-name: com.mysql.cj.jdbc.Driver
    username: root
    password: root

  thymeleaf:
    cache: false

  freemarker:
    cache: false

  flyway:
    enabled: false
    encoding: UTF-8
    sql-migration-prefix: V
    sql-migration-separator: __
    sql-migration-suffixes: .sql
    locations: classpath:db/migration

在代码中使用配置属性

有如下自定义配置

amazon:
  associateId: lyp001

在Java代码中，使用注解@ConfigurationProperties开启配置属性

@Component
@ConfigurationProperties(prefix="amazon")
public class AmazonProperties
{
    private String associateId;

    public String getAssociateId()
    {
        return associateId;
    }

    public void setAssociateId(String associateId)
    {
        this.associateId=associateId;
    }
}

只需要在需要使用该配置属性的地方，将这个 AmazonProperties 注入即可

使用Profile配置

@Profile

被该注解标记的配置类，将在特定profile激活时生效

@Profile(value={"dev","test"})
@Configuration
@EnableWebSecurity
public class SecurityConfig extends WebSecurityConfigurerAdapter
{

}

配置文件命名

  • application-{profile}.properties
  • application-{profile}.yml

激活配置文件,在默认配置文件中激活

spring:
  profiles:
    active: dev

定制错误页面

spring-boot自动配置的默认错误处理器会查找名为error的视图.

只需要根据模板引擎创建 error.xxx 的模板即可

  • thymeleaf - html
  • freemaker - ftl
  • velocity - vm
  • jsp - jsp

spring-boot会为错误视图提供如下错误属性

  • timestamp - 时间点
  • status - http状态码
  • path - url路径
  • error - 错误原因
  • message - 异常消息
  • exception
  • errors
  • trace
