中间件技术-zookeeper安装与部署

笔记 : 中间件技术
发布于 : 2022-06-01

Zookeeper： A Distributed Coordination Service for Distributed Applications（分布式应用程序中的分布式协调服务）。

下载安装

下载：https://zookeeper.apache.org/releases.html

安装：解压压缩包。

单机模式启动

0.修改配置文件 zoo.cfg

  • tickTime：心跳间隔时间，单位ms。session超时时间是2倍的心跳间隔时间
  • dataDir：内存数据库存储位置
  • clientPort：客户端连接端口
tickTime=2000
dataDir=D:/Projects/study/apps/temp/zookeeper
clientPort=2181

1.启动zookeeper服务器

./bin/zkServer.sh start   # linux
.\bin\zkServer.cmd        # windows

2.启动客户端，并连接到服务器

./bin/zkCli.sh -server 127.0.0.1:2181   # linux
.\bin\zkCli.cmd -server 127.0.0.1:2181  # windows

集群模式启动

单机启动多个独立的（非集群）

分别配置 clientPort 和 admin.serverPort，以及data与log目录

dataDir=path/to/data
dataLogDir=path/to/log
clientPort=2181
admin.serverPort=8080
