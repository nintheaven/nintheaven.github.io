Java生成图片验证码

: Java应用篇
发布于 : 2022-10-11

项目地址：https://github.com/lypzzz/lyp-captcha

使用示例

// 创建验证码对象
// 查看CaptchaType枚举类源码了解支持的更多验证码类型
LypCaptcha lypCaptcha = new LypCaptcha(); // 默认为4个字母+数字字符
// LypCaptcha lypCaptcha = new LypCaptcha(4, CaptchaType.CHINESE); // 可以指定字符数和字符类型

// (可选)设置宽高,默认 150*40
lypCaptcha.setSize(200, 50);

// (可选)设置字体,默认 “DejaVu Sans Mono”
// 使用已安装的字体
Font font = new Font("微软雅黑", Font.BOLD, 40);
// 从字体文件创建字体
//Font font = lypCaptcha.loadFontFile("E:\\others\\fonts\\PangMenZhengDaoCuShuTi-2.ttf");
lypCaptcha.setFont(font);

// 纯文本验证码
System.out.println(lypCaptcha.text());

// 字节流验证码
ByteArrayOutputStream image = lypCaptcha.image();
// 也可以创建图片文件
image.writeTo(new FileOutputStream("captcha.png"));
image.close();

// Base64验证码
System.out.println(lypCaptcha.base64());

思路以及关键代码

定义验证码支持的字符类型

要设计验证码要支持的字符类型，当然是使用枚举类来定义

public enum CaptchaType {
    NUMBER,                 // 数字
    CHINESE,                // 中文
    CHINESE_TRADITIONAL,    // 繁体中文
    ENGLISH,                // 大小写英文
    ENGLISH_LOWER,          // 小写英文
    ENGLISH_UPPER,          // 大写英文
    ENGLISH_NUMBER,         // 大小写英文+数字
    ENGLISH_LOWER_NUMBER,   // 小写英文+数字
    ENGLISH_UPPER_NUMBER,   // 大写英文+数字
}

生成每种字符类型的验证码字符串

根据验证码的类型和字符数量，来生成对应的字符串。

public String generate(int size, CaptchaType type) {
    StringBuilder sb = new StringBuilder();
    switch (type) {
        case NUMBER:break;
        case CHINESE:break;
        case CHINESE_TRADITIONAL:break;
        case ENGLISH:break;
        case ENGLISH_LOWER:break;
        case ENGLISH_UPPER:break;
        case ENGLISH_NUMBER:break;
        case ENGLISH_LOWER_NUMBER:break;
        case ENGLISH_UPPER_NUMBER:break;
        default:
            throw new AssertionError("验证码类型不支持");
    }
    return sb.toString();
}

每一个case块中，就是生成对应类型字符串的逻辑。

不管是英文字母、数字还是中文，都可以用Unicode码表中的一个数字来表示，而且它们在Unicode码表中都有自己的范围。利用这点，就可以随机获取1个英文字母，或者数字，或者中文字符

/*随机返回1个给定Unicode编码范围内的字符*/
private char randomChar(int start, int end) {
    return (char) (random.nextInt(end - start + 1) + start);
}

下面是各个类型的字符在Unicode码表中的范围

字符类型 范围(16进制表示)
数字 0x30 - 0x39
英文字母（小写） 0x61 - 0x7a
英文字母（小写） 0x41 - 0x5a
中文字符 0x4e00 - 0x9fa5

对于数字和英文字母来说，直接使用上面的 randomChar(int, int) 方法即可获取到1个随机字符，而中文有点不同，因为简体中文和繁体中文处于同一unicode字符区间，是混到一起的。所以，获取到1个随机中文字符后，需要想办法判断其是否为简体。

办法就是使用只支持简体中文的编码格式（GB2312），对字符进行编码，然后再解码，如果对字符进行编码后还能解码回原来的字符，说明它是简体中文。对于繁体，可以简单的认为处于中文范围内的Unicode都是繁体，或者使用 Big5 做与简体中文一样的操作。

/*随机获取1个中文字符(简体): 0x4e00,0x9fa5*/
private char randomSimpleHans() {
    char c = randomChar(0x4e00, 0x9fa5);
    String cs = String.valueOf(c);
    Charset gb2312 = Charset.forName("GB2312");
    String cs2 = new String(cs.getBytes(gb2312), gb2312);
    if (cs.equals(cs2)) {
        return c;
    } else {
        return randomSimpleHans();
    }
}

既然可以获取到某种类型的单个字符，那么对于多个字符，就很简单的用循环搞定！

绘制验证码图片

Java AWT API 种的 BufferedImage 可以在内存种创建1张图片

BufferedImage image = new BufferedImage(imageW, imageH, BufferedImage.TYPE_INT_RGB);

然后，它的 getGraphics() 方法返回1个 Graphics 对象，使用这个对象，就可以进行图形绘制了

Graphics g = image.getGraphics();

首先，绘制1个矩形，用于设置图片的尺寸范围

g.fillRect(0, 0, imageW, imageH);

然后，在图片上绘制一些干扰点

/*绘制干扰点*/
private void drawInterferencePoints(Graphics g, int imageW, int imageH) {
    int x, y;
    for (int i = 0; i < INTERFERENCE_POINT_NUM; i++) {
        x = random.nextInt(imageW);
        y = random.nextInt(imageH);
        g.setColor(randomColor());
        g.fillOval(x, y, random.nextInt(5) + 1, random.nextInt(5) + 1);
    }
}

接下来，将验证码字符串绘制上去

/*绘制验证码*/
private void drawCaptchaCode(Graphics g, String code, Font font, int imageW, int imageH) {
    g.setFont(font);
    int fontH = getFontHeight(font); // 字体高度
    int size = code.length(); // 验证码字符数

    int x;
    int y = imageH / 2 + fontH / 4;
    for (int i = 0; i < size; i++) {
        g.setColor(randomColor()); // 随机颜色
        g.translate(random.nextInt(3), random.nextInt(3));
        x = i * (imageW / size);
        g.drawString(String.valueOf(code.charAt(i)), x, y);
    }
}

/*获取字体高度*/
private int getFontHeight(Font font) {
    FontDesignMetrics metrics = FontDesignMetrics.getMetrics(font);
    return metrics.getHeight();
}

最后，要记得调用 dispose() 方法释放系统资源

g.dispose();

输出图片

经过上面的操作，验证码图片已经绘制完成了，只需要使用 ImageIO 的 write(RenderedImage, String, OutputStream) 静态方法将 RenderedImage 写入一个 ByteArrayOutputStream 中，这个输出流就是最终的验证码图片字节流

// 将图片数据写入到到字节输出流中
try (ByteArrayOutputStream out = new ByteArrayOutputStream();) {
    boolean success = ImageIO.write(image, "png", out);
    if (success) {
        return out;
    }
} catch (IOException e) {
    e.printStackTrace();
}

要想创建图片文件，可以使用 ByteArrayOutputStream 的 writeTo(OutputStream) 方法

// 创建图片文件
ByteArrayOutputStream image = ...;
image.writeTo(new FileOutputStream("captcha.png"));

要想获取base64格式的图片数据，只需要从字节输出流中获取到字节数组，然后使用 Base64.Encoder 对对字节数组进行编码即可

ByteArrayOutputStream out = ...;
Base64.Encoder encoder = Base64.getEncoder();
return encoder.encodeToString(out.toByteArray());

img标签使用base64图片数据时，要在base64字符串前面拼上

data:image/png;base64,

结语

Java图片验证码的思路还是简单的，熟悉Java AWT API的话，很容易就可以实现一个简单的验证码工具库。
