SpotBUGs

: Java应用篇
发布于 : 2022-10-11

官网： https://spotbugs.github.io/

添加maven插件

<plugin>
    <groupId>com.github.spotbugs</groupId>
    <artifactId>spotbugs-maven-plugin</artifactId>
    <version>4.2.0</version>
    <executions>
        <execution>
            <id>check</id>
            <phase>test</phase>
            <goals>
                <goal>check</goal>
            </goals>
        </execution>
    </executions>
    <dependencies>
        <dependency>
            <groupId>com.github.spotbugs</groupId>
            <artifactId>spotbugs</artifactId>
            <version>4.2.3</version>
        </dependency>
    </dependencies>
</plugin>

使用

mvn compile

mvn spotbugs:check
