Java中的实体类

: Java应用篇
发布于 : 2022-10-11

POJO (Plain Old Java Object)

简单的老式Java对象，简单而纯粹，只包含自身private属性和getter,setter方法。

PO (persistent object)

持久层对象，在ORM框架中与数据表对应。

DTO (Data Transfer Object)

数据转移(传输)对象，用于在不同层(例如控制层与服务处)之间交换数据。

VO (Value Object)

最终给前端展示的数据对象。

BO (business object)

业务对象，类似于DTO，但是包含复杂的业务逻辑。
