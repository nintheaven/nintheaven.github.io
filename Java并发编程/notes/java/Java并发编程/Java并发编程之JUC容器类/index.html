Java并发编程中的容器类

2022-10-11

Vector

HashTable

Vector是线程安全的ArrayList，HashTable是线程安全的HashMap。

它门的原理是所有读写操作方法都是同步方法，保证了线程安全。

Collections

Collections中提供了静态的方法，用于将List，Set，Map转换为线程安全的容器类型，其原理是将读写操作放到静态代码块中。

使用了装饰器设计模式。

public class Collections {
    public static <T> List<T> synchronizedList(List<T> list){
        return (list instanceof RandomAccess ?
                new SynchronizedRandomAccessList<>(list) :
                new SynchronizedList<>(list));
    }

    public static <T> Set<T> synchronizedSet(Set<T> s) {
        return new SynchronizedSet<>(s);
    }

    public static <K,V> Map<K,V> synchronizedMap(Map<K,V> m) {
        return new SynchronizedMap<>(m);
    }
}

CopyOnWriteArrayList

CopyOnWriteArraySet

写时复制技术。
读的时候不加锁，写的时候使用同步代码块加锁，并且复制原数组到一个新的数组，修改新数组，然后用新数组替换原来的数组。

CopyOnWriteArraySet内部使用CopyOnWriteArrayList实现(装饰器设计模式)。
