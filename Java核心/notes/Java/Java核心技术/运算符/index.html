Java核心-运算符

笔记 : Java核心
发布于 : 2021-02-01

赋值

赋值运算符是=，将右边的值赋给左边的变量。

对于 a=b：

  • b是基本类型，直接复制一份b的值给a
  • b是对象，赋予a的只是b的引用的一份拷贝；

在方法调用中，参数传递也是一样的策略。

算术运算符

加+、减-、乘*、除/、模%。

一元加减运算符（-1、+1），一元加运算符的唯一的副作用是把较小范围的整数转换位int类型。

算术运算符可以和赋值运算符进行组合：+=、-=、*=、/=、%=

递增递减

前递增、后递增，前递减、后递减。

关系运算符

大于>、小于<、大于等于>=、小于等于<=、等于==、不等于!=。

需要注意的是，比较对象是否相等，应该使用 equals() 方法。

逻辑运算符

与&&、或||、非!。

逻辑运算符支持“短路”操作：

  • a && b && c，当前面的表达式值位false时，整个表达式直接返回false，后面的表达式不会执行

  • a || b || c，当前面的表达式值为true时，整个表达式直接返回true，后面的表达式不会执行
