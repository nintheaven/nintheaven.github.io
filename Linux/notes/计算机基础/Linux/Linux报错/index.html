Linux报错

Linux 2022-10-11

一次配置环境变量引起的小事故

部署应用到客户的centos服务器，配置Java环境变量

vi /etc/profile

手滑把$PATH写成了$APTH，然后执行了source命令。结果导致常用命令都用不了，包括vi。

因为只能通过网页版的控制台操作，无法使用xshell等图形工具远程连接，紧张了一把。。。

搜索“linux服务器 配置环境变量语法错误，导致vim命令无法使用”，结果很nice

export PATH=$PATH:/usr/local/sbin:/usr/local/bin:/sbin:/bin:/usr/sbin:/usr/bin

幸好export还没受到影响。。

cat /var/spool/mail/root
