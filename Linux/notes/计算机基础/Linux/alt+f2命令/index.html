alt+f2命令

笔记 : Linux
发布于 : 2022-10-11

alt+f2应该是linux桌面中最好用的快捷键了。

常用命令

启动命令行：gnome-terminal

启动VSCode：code

自定义命令

alt+f2命令会从PATH环境变量中查找可执行程序。在ubuntu中，~/.profile 文件内容如下

# ~/.profile: executed by the command interpreter for login shells.
# This file is not read by bash(1), if ~/.bash_profile or ~/.bash_login
# exists.
# see /usr/share/doc/bash/examples/startup-files for examples.
# the files are located in the bash-doc package.

# the default umask is set in /etc/profile; for setting the umask
# for ssh logins, install and configure the libpam-umask package.
#umask 022

# if running bash
if [ -n "$BASH_VERSION" ]; then
    # include .bashrc if it exists
    if [ -f "$HOME/.bashrc" ]; then
	. "$HOME/.bashrc"
    fi
fi

# set PATH so it includes user's private bin if it exists
if [ -d "$HOME/bin" ] ; then
    PATH="$HOME/bin:$PATH"
fi

# set PATH so it includes user's private bin if it exists
if [ -d "$HOME/.local/bin" ] ; then
    PATH="$HOME/.local/bin:$PATH"
fi

可以发现，除了会加载 ~/.bashrc 中的配置外，还会从 ~/bin/ 和 ~/.local/bin 中查找可执行的命令。

所以，只要在这两个目录中的可执行脚本文件，都可以使用 alt+f2 快速执行。

以启动idea为例，idea安装在 /opt/ 目录下，操作如下：

cd ~/.local/bin

touch idea

vim idea

内容如下

#!/bin/bash

cd /opt/idea-IU-212.5284.40/bin/
nohup /imagesdea.sh &

chmod +x idea

注销，登录后，就可以使用 alt+f2 快捷键启动 idea 了。
