Linux基本设置

笔记 : Linux
发布于 : 2022-10-11

设置时区

tzselect

设置日期时间

date -s 11/03/17

date -s 14:20:50

设置系统语言

查看当前系统语言：locale

查看全部系统语言：locale -a

修改 /etc/locale.conf，设置LANG变量

#LANG=en_US.UTF-8
LANG=zh_CN.UTF8

使其生效

source /etc/locale.conf
