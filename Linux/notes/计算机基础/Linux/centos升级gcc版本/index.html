centos升级gcc版本

笔记 : Linux
发布于 : 2022-10-11

centos7中，gcc默认的版本是4.8.5。

升级到gcc8+

1.安装scl

yum install centos-release-scl

2.安装gcc工具集

yum install devtoolset-8-gcc*

会安装在/opt/rh/devtoolset-8目录下

3.激活

一次性生效

scl enable devtoolset-8 bash

或者

source /opt/rh/devtoolset-8/enable

永久生效：替换默认的gcc和g++

默认的gcc以及g++在/usr/bin/ 目录下，只需要用/opt/rh/devtoolset-8/root/ 目录下的gcc和g++进行替换即可永久生效。推荐使用软连接。
