centos用户与权限

Linux 2022-10-11

创建用户

useadd命令使用 -m 选项，自动创建home目录，并将 /etc/skel/ 中的文件复制进来

useadd -m test

ls -al /home/test/

删除用户

userdel命令使用 -r 选项，删除用户的同时，删除其在home目录下的文件

userdel -r test

修改密码

passwd

密码过于简单时，会修改失败

使用下面的命令即可

echo <username> | passwd --stdin <newPassword>

非root用户执行sudo命令时报错

原因是yyds用户不在 /etc/sudoers 配置文件中。只需要在此配置文件添加yyds用户或所属组即可。
