ssh,scp,ftp

2022-10-11

ssh

连接远程主机

ssh 主机名@主机ip

scp

复制本地文件到远程主机

复制文件

  • scp 本地文件名 远程主机用户名@远程主机ip:远程目录/

  • scp 本地文件名 远程主机用户名@远程主机ip:远程文件名

  • scp 本地文件名 远程主机ip:远程目录/

  • scp 本地文件名 远程主机ip:远程文件名

复制目录

使用 -r 参数

  • scp -r 本地目录 远程主机用户名@远程主机ip:远程目录

复制远程主机文件到本地

  • scp 远程主机用户名@远程主机ip:远程文件名 本地文件名

  • scp -r 远程主机用户名@远程主机ip:远程目录 本地目录
