配置https服务器

安装ssl模块

检查ssl模块是否已安装

nginx -V

如果有 –with-http_ssl_module 选项，则已安装，否则，进行以下操作。

来到nginx源码包路径下

cd /opt/nginx-1.9.5

sudo ./configure --with-http_ssl_module

sudo make
