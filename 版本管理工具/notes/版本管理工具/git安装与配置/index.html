git安装与配置

笔记 : 版本管理工具
发布于 : 2021-12-01

git安装

linux

sudo apt install git

windows

直接下载exe文件进行安装。

git配置

git config --global user.name xxx

git config --global user.email xxx

在远程仓库设置sshkey

运行命令 ssh-keygen ，将在 .ssh 目录下生成公钥和私钥文件，复制公钥中的内容到远程仓库即可。

git push 报错

fatal: unable to access 'https://github.com/lypzzzzz/blog-repo.git/': OpenSSL SSL_read: Connection was reset, errno 10054

之所以报此错误，是因为https证书没有在相关机构进行过签署。只需要进行以下配置，在push的时候，不进行ssl认证即可

git config --global http.sslverify "false"
