FreeMarker入门

: HTML模板引擎
发布于 : 2022-10-11 
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-freemarker</artifactId>
    <version>2.5.1</version>
</dependency>

freemarker模板文件扩展名默认为 .ftlh(旧版为.ftl)。

开发阶段禁用模板缓存 

spring: 
  freemarker:
    cache: false

更改模板内容后，build(ctrl+f9)，模板会自动更新
