功能介绍

基于SpringBoot+Vue实现的前后端分离个人博客系统。

实现了管理员登录、博客文章、读书笔记、个人作品集、网址收藏、以及站点信息管理等功能。

技术栈

  • 后端：SpringBoot, MyBatis, Swagger, MySQL, Redis, JWT
  • 前端：Vue, VueRouter, ElementUI, Axios
  • 环境：Ubuntu, Nginx, 腾讯云服务器

项目结构

项目整体划分为3个子模块：

API服务端App

  • API服务端总体划分为后台API与前台API两个模块

  • API服务对外提供一致的REST Full风格的API接口，并使用Swagger自动生成API文档

  • 使用JWT方案，结合Spring拦截器以及自定义注解实现用户身份验证功能

  • 自定义验证码工具类，为以Base64字符串的形式返回给客户端

  • 使用MyBatis进行数据库操作，并使用Flyway进行SQL版本管理

Admin博客后台管理系统

  • 博客后台管理系统包括博客文章管理、读书笔记管理、个人作品集管理、网址收藏管理、图片管理以及站长信息管理等功能

  • 基于Vue+ElementUI实现

  • 使用VueRouter实现单页应用的路由功能

  • 使用Axios进行HTTP请求

Web端App

  • Web端使用Vue与Axios进行开发
  • 使用浏览器媒体查询以及flex和grid布局实现移动端屏幕适配
