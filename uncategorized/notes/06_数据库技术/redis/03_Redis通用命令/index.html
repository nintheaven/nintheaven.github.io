Redis通用命令

2022-10-11

键命令

keys <pattern>

查找所有符合pattern的key

* 星号匹配0或多个任意字符：keys *

? 问号匹配0或1个任意字符：keys h?llo

[] 方括号匹配字符集中的任意1个字符：keys h[ab]llo keys h[a-z]llo keys h[^a]llo

exists key1 key2 ...

判断给定的key是否存在。

del key1 key2 ...

删除key

type key

查看key的数据类型。

expire key seconds

设置key的过期时间，单位：s

expireat key timestamp

设置key的过期时间，使用绝对 Unix 时间戳 (自1970年1月1日以来的秒数)

ttl key

查看key的过期时间。

rename key newkey

重命名key。

scan cursor

遍历当前数据库中的key

数据库命令

select index

选择数据库。

redis默认有0-15共16个数据库

dbsize

查看当前数据库中共有多少个key

flushdb

清空当前数据库

flushall

清空全部数据库
