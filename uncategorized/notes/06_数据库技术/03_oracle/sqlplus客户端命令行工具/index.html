sqlplus客户端命令行工具

2022-10-11

以 sysdba 的身份登录Oralce

sqlplus / as sysdba

sqlplus system/password@instance_name

使用用户名和密码登录Oracle

sqlplus username/password@instance_name

查看，设置输出结果中每行的大小。默认80

show linesize;

set linesize 150;

查看，设置输出结果中每页的行数。默认14

show pagesize;

set pagesize 30;

查看当前登录的数据库名

show parameter db_name;

退出Oracle

quit

END
