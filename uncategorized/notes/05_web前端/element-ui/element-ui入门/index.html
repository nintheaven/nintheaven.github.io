2022-10-11

环境搭建

安装

npm install element-ui --save

引入Element UI组件

一次性引入(不推荐)

按需引入(推荐)

安装 babel-plugin-component

npm install babel-plugin-component --save-dev

然后在vue项目根目录下的 .babelrc 文件中(没有就新建)进行以下配置

报错

安装 @babel/preset-env

npm install @babel/preset-env --save-dev

然后把 .babelrc 配置中的 es2015 替换为 @babel/preset-env

实际上，vue项目中可以不用新建 .babelrc 文件的，直接在 babel.config.js 中配置即可

接下来，按需引入组件即可

最佳实践

如果全部组件都在main.js中进行import，那会把main.js弄得乱七八糟。推荐使用插件的方式按需引入element-ui组件。

首先在 src 目录下新建 plugins/element/index.js 文件，在这个文件中import组件

然后在 main.js 中使用它

Vue.use()的时候，会自动执行插件中的 install 方法。
