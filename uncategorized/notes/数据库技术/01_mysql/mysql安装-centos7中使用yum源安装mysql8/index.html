mysql安装-centos7中使用yum源安装mysql8

: 未分类
发布于 : 2022-10-11

使用yum安装MySQL

参考文档： https://dev.mysql.com/doc/refman/8.0/en/linux-installation-yum-repo.html

1.下载yum仓库的rpm文件

https://dev.mysql.com/downloads/repo/yum/

2.安装上面下载的rpm文件

sudo yum install mysql80-community-release-el7-5.noarch.rpm

3.安装mysql

sudo yum install mysql-community-server

4.启动mysql服务

systemctl start mysqld

systemctl status mysqld

5.修改root密码

查看安装时自动设置的临时密码

sudo grep 'temporary password' /var/log/mysqld.log

使用临时密码登录mysql

mysql -u root -p

更改root密码

ALTER USER 'root'@'localhost' IDENTIFIED BY 'MyNewPass4!';

卸载mysql

1.查看MySQL相关的包

rpm -qa | grep mysql

2.使用yum remove命令移除掉所有的包

3.使用find命令找到mysql相关的目录与文件，然后全部rm

find / -name mysql
