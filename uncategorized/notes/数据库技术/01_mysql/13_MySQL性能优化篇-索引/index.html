索引基础

定义：索引(MySQL中也叫做键Key)，是存储引擎用于快速找到记录的一种数据结构。

MySQL中的索引(index)使用B+树实现的，树中的每一个节点都指向对应的数据行。

MySQL先在索引上按值进行查找，然后返回所有包含该值的数据行。

优点

查询，排序操作通过B+树，效率比较高；

缺点

占用一定的存储空间；

更新表(insert,update,delete)数据时，同时要维护索引，会降低效率；

维护索引增加了数据库维护人员的工作量。

索引类型

  • 单值索引
  • 复合索引
  • 唯一索引(unique)，主键自动建立唯一索引

索引SQL

创建索引

create index 索引名 on 表名 (索引列,...);
create unique index 索引名 on 表名 (索引列,...);

修改索引

删除索引

drop index 索引名 on 表名;

查询索引

show index from 表名;
