算术运算符

运算符 说明 示例
+ 加法 1000+1000
- 减法 1000-1000
* 乘法 1000*3
/ 或 DIV 除法，推荐使用除号(/)，而不是MySQL方言DIV 1000/3 或 1000 DIV 3
% 或 MOD 取模，推荐使用百分号(%)，而不是MySQL方言MOD 1000%3 或 1000 MOD 3

例如，对员工的基础工资分别进行加、减、乘、除、取模运算

mysql> SELECT basic,basic+1000,basic-1000,basic*3,basic/3,basic%3 FROM Salary;

比较运算符

运算符 说明 示例
= 等于 age=20
<> 或 != 不等于 age<>20
> 大于 age>20
>= 大于等于 age>=20
< 小于 age<20
<= 小于等于 age<=20

需要注意的是，NULL 参与的比较运算，结果都是NULL

mysql> SELECT 1=NULL,1<>NULL,1>NULL,1<NULL;
+--------+---------+--------+--------+
| 1=NULL | 1<>NULL | 1>NULL | 1<NULL |
+--------+---------+--------+--------+
|   NULL |    NULL |   NULL |   NULL |
+--------+---------+--------+--------+

逻辑运算符

AND 优先级高于 OR，可以使用圆括号()改变优先级。

运算符 说明 示例
NOT 逻辑非，对布尔表达式取反 NOT last_name=’李’
AND 逻辑与，对布尔表达式进行与运算 age<20 AND sex=’female’
OR 逻辑或，对布尔表达式进行或运算 (age<20 AND sex=’female’) OR (age>30 AND sex=’male’)

处理NULL参与的运算

对于 NULL 参与的运算，要使用 IS NULL 或 IS NOT NULL。
