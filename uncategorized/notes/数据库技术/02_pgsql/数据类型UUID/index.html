安装uuid-ossp扩展

安装依赖包

uuid-ossp扩展依赖 e2fsprogs-devel, uuid-devel, libuuid-devel 3个依赖包

下载地址：https://developer.aliyun.com/packageSearch

安装依赖包

rpm -Uvh --force *.rpm

如果可以连接网络，直接用yum安装

yum -y install e2fsprogs-devel uuid-devel libuuid-devel

编译安装uuid-ossp包

cd /opt/postgresql-13.6

./configure --prefix=/usr/local/pgsql --with-uuid=ossp

cd contrib/uuid-ossp

make & make install

此时，可以在pgsql安装目录下的lib文件夹中生成了uuid-ossp.so文件

ls /usr/local/pgsql/lib

创建uuid-ossp扩展插件

CREATE extension IF NOT EXISTS "uuid-ossp";

使用uuid

SELECT uuid_generate_v4();
SELECT gen_random_uuid();

建表的时候，主键设置为UUID类型，默认值使用 uuid_generate_v4() 函数。

CREATE TABLE xxx (
  "id" uuid NOT NULL DEFAULT uuid_generate_v4(),
);
