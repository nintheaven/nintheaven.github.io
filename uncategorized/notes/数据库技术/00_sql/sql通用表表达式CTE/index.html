sql通用表表达式CTE

2022-10-11

CTE(Common Table Expression，通用表表达式)。

WITH 
    cte_a AS 
        SELECT a,b FROM ...,
    cte_b AS 
        SELECT x,y FROM ...
SELECT a,b,x,y FROM cte_a JOIN cte_b ON ...;

显式指定CTE中的列名

WITH 
    cte_a (a, b) AS 
        SELECT a,b FROM ...,
    cte_b (x, y) AS 
        SELECT x,y FROM ...
SELECT a,b,x,y FROM cte_a JOIN cte_b ON ...;
END
