参考约束

数据库并不会强制执行参考约束，参考约束的作用是提高数据库的查询性能。

check约束

check约束用于限制某些列的值。

mysql虽然可以使用check约束，但是并不强制执行。

gender CHAR(1) CHECK (gender IN ('M','F'))

字符集

mysql

查看支持的字符集

SHOW CHARACTER SET;

默认字符集

mysql8之前为latin1，版本8开始为utf8mb4。

为数据库设置字符集

CREATE DATABASE test CHARACTER SET utf8mb4;

为数据表设置字符集

CREATE TABLE IF NOT EXISTS t_persoon (
  id int(11) NOT NULL
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;

为列指定列字符集

first_name VARCHAR(20) CHARACTER SET utf8mb4,
last_name VARCHAR(20) CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci,
