创建微服务项目

只有2个微服务：商品(product)和订单(order)。

商品微服务

实体类

@Data
@NoArgsConstructor
@AllArgsConstructor
public class Product {
    private Long id;
    private String name;
    private BigDecimal price;
    private Integer inventory;
}

service层

这里没有创建数据库，而是使用假数据进行模拟。

@Service
public class ProductService {
    // 模拟数据库
    private static Map<Long,Product> db = new HashMap<Long,Product>(){{
        put(1L,new Product(1L,"联想Yoga14S", BigDecimal.valueOf(6899.00),99));
        put(2L,new Product(2L,"kindle paperwhite5", BigDecimal.valueOf(1300.00),9));
        put(3L,new Product(3L,"樱桃(CHERRY)有线鼠标", BigDecimal.valueOf(76),100));
        put(4L,new Product(4L,"任天堂 Switch", BigDecimal.valueOf(2000.00),0));
    }};

    public Product findById(Long productId){
        return db.get(productId);
    }
}

controller层

商品服务运行在8000端口，只有一个方法，用于返回给定ID的商品信息

@RestController
public class ProductController {
    @Autowired
    private ProductService productService;

    @GetMapping("/product")
    public ResponseEntity<Product> get(@RequestParam("productId")Long productId){
        Product product = productService.findById(productId);
        if (product == null) {
            return ResponseEntity.notFound().build();
        }
        return ResponseEntity.ok(product);
    }
}

订单微服务

实体类

@Data
@NoArgsConstructor
@RequiredArgsConstructor
public class Order {
    @NonNull
    private Long id;
    @NonNull
    private List<Long> productIds;
    
    private List<Product> products;
}

service层

这里没有创建数据库，而是使用假数据进行模拟。

@Service
public class OrderService {
    // 模拟数据库
    private static Map<Long, Order> db = new HashMap<Long,Order>(){{
        put(1L,new Order(1L, Arrays.asList(1L)));
        put(2L,new Order(1L, Arrays.asList(1L,2L)));
        put(3L,new Order(1L, Arrays.asList(2L,3L,4L)));
    }};

    public Order findById(Long orderId){
        return db.get(orderId);
    }
}
