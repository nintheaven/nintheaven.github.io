集成editormd

: Web前端开发
发布于 : 2022-10-11

https://pandao.github.io/editor.md/

下载文件后，将需要用到的文件复制到项目的相关目录下,目录结构如下

添加编辑器

<html>
<head>
    <link rel="stylesheet" type="text/css" href="/editormd/css/editormd.min.css" />
</head>
<body>
    <div id="editor">
        <textarea name="markdown" style="display:none;"></textarea>
    </div>

    <script type="text/javascript" src="/js/jquery.min.js"></script>
    <script type="text/javascript" src="/editormd/editormd.min.js"></script>

    <script type="text/javascript">
        $(function() {
            var editor = editormd("editor", {
                width: "100%",
                height: "100vh",
                markdown: "",     // 可以使用js动态设置markdown的内容
                path : "/editormd/lib/"  // Autoload modules mode, codemirror, marked... dependents libs path
            });
        });
    </script>
</body>
</html>

图片上传

在js中添加以下配置选项

imageUpload          : true,          // 开启本地图片上传功能
imageFormats         : ["jpg", "jpeg", "gif", "png", "bmp", "webp"],
imageUploadURL       : "/admin/articles/image/upload",             // 处理图片上传的后端接口URL
crossDomainUpload    : false,          // Enable/disable Cross-domain upload
uploadCallbackURL    : "",             // Cross-domain upload callback url

图片表单字段的name为 editormd-image-file 后端控制器中，接收图片数据

@PostMapping("/image/upload")
@ResponseBody
public ResponseEntity<Map<String,Object>> doUploadImage(@RequestParam("editormd-image-file") MultipartFile image)
{
    String imageUrl=null;
    // 保存图片到本地或云端,返回图片url
    // ...

    Map<String,Object> map=new HashMap<>();
    if(imageUrl!=null)
    {
        map.put("success",1);
        map.put("message","上传成功");
        map.put("url",imageUrl);
    }
    else
    {
        map.put("success",0);
        map.put("message","上传失败");
    }
    return ResponseEntity.ok(map);
}

响应为体格式

{
    success : 0 | 1,           // 0 表示上传失败，1 表示上传成功
    message : "提示的信息，上传成功或上传失败及错误信息等。",
    url     : "图片地址"        // 上传成功时才返回
}

报错403

Failed to load resource: the server responded with a status of 403 ()
chrome-error://chromewebdata/:1 Refused to display 'http://localhost:8080/' in a frame because it set 'X-Frame-Options' to 'deny'.

是因为使用了spring-security，在SercurityConfig中，添加以下配置

@Override
protected void configure(HttpSecurity http) throws Exception
{
    http.headers().frameOptions().disable();
}

依然报错403

Uncaught SyntaxError: Unexpected token F in JSON at position 14

是因为spring-security开启了csrf保护，表单提交需要token，但是editor.md中没有，解决办法就是对上传图片的url关闭csrf保护

@Override
protected void configure(HttpSecurity http) throws Exception
{
    http.headers().csrf().ignoringAntMatchers("/admin/articles/image/upload")
}
