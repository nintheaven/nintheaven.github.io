JavaScript-加密与解密

Web前端开发 2022-10-11

md5加密

开源工具库：https://github.com/blueimp/JavaScript-MD5

引入

<script src="js/md5.min.js"></script>

或者npm

npm install blueimp-md5 --save

使用

import md5 from "blueimp-md5"

var hash = md5('value') // "2063c1608d6e0baf80249c42e2be5804"
END
