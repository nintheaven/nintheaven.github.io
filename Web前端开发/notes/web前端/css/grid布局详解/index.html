grid布局详解

2022-10-11

grid布局内容过宽时撑开网格的问题

需要在父容器中，使用overfolow: hidden，然后在子布局中使用overflow:auto，让子网格超出其宽度时滚动

.wrapper {
    width: 80%;
    margin: 0 auto;
    display: grid;
    grid-template-columns: 1fr 10fr;
    grid-auto-rows: minmax(2rem, auto);
    grid-gap: 2rem;
    gap: 2rem;
    overflow:hidden
}

.aside {
    grid-column: 1;
    overflow: auto;
}

.main {
    grid-column: 2;
    overflow: auto;
}
END
