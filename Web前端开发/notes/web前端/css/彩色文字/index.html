彩色文字

: Web前端开发
发布于 : 2022-10-11

html

<div>
    <span class="word">THINK</span>
    <span class="word">PLAN</span>
    <span class="word">ACTION</span>
</div>

css

.word{
    font-weight: 800;
    font-size: 3rem;
    padding: 1rem;
    letter-spacing: 10px;
    color: transparent; /* 文字设置为透明 */
    background: linear-gradient(to right, #1fbae6, #f8062e); /* 背景设置为彩色的渐变色 */
    -webkit-background-clip: text; /* 裁剪背景,只保留文字部分 */
}
