层叠和继承

2022-10-11

层叠规则

层叠规则是当声明冲突时，CSS解决冲突的规则。规则如下：

1.根据样式表的来源

按照样式表的来源，重要程度从低到高依次为：浏览器默认样式，外部CSS文件，内部style标签，内联样式。

不推荐使用行内样式。

2.根据选择器的优先级

CSS选择器的优先级决定于选择器类型和数量，同样数量的情况下，优先级从高到低依次为：ID选择器，class选择器，标签选择器。

伪类选择器和属性选择器与class选择器优先级相同；

通用选择器（*）和组合选择器（> + ~）对优先级没有影响；

!important 修饰的规则拥有最高的优先级。

不推荐使用ID选择器或者使用 !important。

3.根据规则声明的先后顺序

最后声明的规则生效。

2个特殊值

inherit 继承父元素的属性。

initial 恢复默认值。
