nodejs与npm环境搭建

2022-10-11

nodejs安装与PATH环境变量的配置；npm配置与常用命令。

下载与安装

下载

  • http://nodejs.cn/download/current/
  • https://nodejs.org/dist/
  • https://registry.npmmirror.com/binary.html?path=node/

windows

安装

安装目录为 D:\dev\nodejs\

配置PATH环境变量

在系统变量中的PATH变量中添加nodejs安装路径 *D:\dev\nodejs\bin* ，这样就可以使用 node 和 npm , npx 命令了。

node -v

npm -v

最好在安装完成后，修改nodejs安装目录的权限，允许当前用户进行读写操作，不然可能会有各种权限问题。。

Linux

下载压缩包，解压即可。然后配置PATH环境变量，设置模块安装路径等与Windows类似。

NPM

文档：http://nodejs.cn/learn/an-introduction-to-the-npm-package-manager

配置

全局安装模块路径与缓存路径

默认情况下，npm全局安装的模块在 *C:\Users\li\AppData\Roaming\npm* 目录下的 node_modules 文件夹下，推荐修改为nodejs的安装路径下。

确保有读写权限。

linux下使用 chmod 和 chown 命令修改权限和属组

在nodejs安装目录下新建 node_global 和 node_cache 文件夹，然后执行以下npm命令进行设置

npm config set prefix "D:\dev\nodejs\node_global"

npm config set cache "D:\dev\nodejs\node_cache"

这样，使用npm全局安装命令安装的模块会被安装到 node_global 目录下的 node_modules 文件夹中。

然后，新建系统环境变量 NODE_PATH ,值为 D:\dev\nodejs\node_global\node_modules。使用命令 npm root -g 查看。

最后，还要将 node_global 目录设置到系统PATH环境变量中：D:\dev\nodejs\node_global。

全局安装的可执行文件路径，例如vue命令。

NPM镜像

默认镜像：https://registry.npmjs.org/

淘宝镜像

  • 临时使用

    npm install <package_name> --registry=https://registry.npm.taobao.org

  • 持久使用

    npm config set registry https://registry.npm.taobao.org

    可以通过命令 npm config get registry 查看已经配置的镜像。

  • 使用cnpm

    npm install -g cnpm --registry=https://registry.npm.taobao.org

常用命令

查看npm的版本号。

npm -v

查看某个命令的帮助文档。

npm help <command_name>

在node项目下运行该命令，npm会安装package.json中所声明的依赖模块，安装到node项目的node_modules目录下。

npm install

在node项目下运行该命令，npm会将单个模块安装到node_modules目录下。

可选的参数：

--save：安装并在package.json的dependencies配置项下进行声明；

--save-dev：安装并在package.json的devDependencies配置项下进行声明。

npm install <package-name>

全局安装，安装到配置的全局安装目录中，可以使用 npm root -g 命令查看具体安装到计算机上的位置。

npm install -g <package-name>

使用@语法，安装特定版本的包（默认安装npm仓库中最新可用版本）。

npm install <package>@<version>

npm install -g <package>@<version>

卸载包。

npm uninstall <package-name>

npm uninstall -g <package-name>

npm uninstall --save <package-name> 或 npm uninstall -S <package-name>

npm uninstall --save-dev <package-name> 或 npm uninstall -D <package-name>

更新全部或单个模块。

npm update

npm update <package-name>

查看本地安装（或者全局安装）的所有包及其版本信息。

npm list

npm list -g

查看本地安装（或者全局安装）的所有包及其版本信息，只列出顶层包。

npm list --depth=0

npm list -g --depth=0

查看软件包在npm仓库中的最新可用版本。

npm view <package_name> version

查看软件包在npm仓库中的全部可用版本。

npm view <package_name> versions

自动修复安装失败的包。

npm audit fix

npm audit fix --force

CNPM

安装

npm install -g cnpm --registry=https://registry.npm.taobao.org

cnpm -v
