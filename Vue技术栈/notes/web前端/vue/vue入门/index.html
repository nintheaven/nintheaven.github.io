vue中引入css文件

方式1：在public/index.html中使用link标签引入

在public目录下新建css文件夹，将css文件放在其中，然后，使用link标签引入

<link rel="stylesheet" href="/css/reset.css">

方式2：在App.vue中使用@import引入src下的css文件

<style>
@import "~@/assets/css/global.css";
</style>

方式3：在main.js中引入src下的css文件

import '@/static/css/global.css'
