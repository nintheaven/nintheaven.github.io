vue项目部署

2022-10-11

vue项目部署后首屏加载过慢问题解决方案

1.删除js文件夹下的map文件

这些文件主要是帮助我们线上调试代码，查看样式。所以为了避免部署包过大，通常都不生成这些文件。

可以在vue.config.js中进行配置，打包时不生成map文件

module.exports = defineConfig({
    transpileDependencies: false
});

2.vue-router路由懒加载

3.开启gzip压缩

没有压缩的情况下,css文件夹下的chunk-vendors.css有300多kb，js文件夹下的chunk-vendors.js有1.7M之多!

服务器带宽没有10M以上，首屏加载不慢才怪!

首先，安装插件

如果最新版报错，就安装较旧的版本

npm i compression-webpack-plugin@5.0.1

然后，再vue中配置gizp压缩，下面是vue.config.js配置文件

const path = require('path');

const webpack = require('webpack')
const CompressionWebpackPlugin = require('compression-webpack-plugin')
const productionGzipExtensions = ['js', 'css']
// const isProduction = process.env.NODE_ENV === 'production'

module.exports = {
    lintOnSave: false,
    runtimeCompiler: true,
    publicPath: './', // 设置打包文件相对路径

    devServer: {
        port: 10001,
        // proxy: {
        //     '^/api': {
        //         target: 'http://localhost:8080',
        //         changeOrigin: true,
        //         ws: false,
        //         pathRewrite: {
        //             //'^/api': '/api/v1'
        //         }
        //     }
        // }
    },

    configureWebpack: {
        resolve: {
            alias: {
                '@': path.resolve(__dirname, './src'),
                '@i': path.resolve(__dirname, './src/assets'),
            }
        },
        plugins: [
            new webpack.IgnorePlugin(/^\.\/locale$/, /moment$/),

            // 下面是下载的插件的配置
            new CompressionWebpackPlugin({
                algorithm: 'gzip',
                test: new RegExp('\\.(' + productionGzipExtensions.join('|') + ')$'),
                threshold: 10240,
                minRatio: 0.8
            }),
            new webpack.optimize.LimitChunkCountPlugin({
                maxChunks: 5,
                minChunkSize: 100
            })
        ]
    }
}

此时，重新打包vue项目，会生成压缩过的.gz文件

最后，在nginx中配置gzip压缩

server{
    listen 80;
    server_name localhost;

    gzip on;
    gzip_min_length 1k;
    gzip_comp_level 9;
    gzip_types text/plain application/javascript application/x-javascript text/css application/xml text/javascript application/x-httpd-php image/jpeg image/gif image/png;
    gzip_vary on;
    gzip_disable "MSIE [1-6]\.";

    location / {
        root xxx;
        index index.html;
        try_files $uri $uri/ /index.html;
    }
}

4.按需引入组件

例如element-ui等，只引入需要用到的组件，而不是图方便全部引入。
