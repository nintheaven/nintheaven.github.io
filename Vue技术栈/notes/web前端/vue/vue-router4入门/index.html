vue-router4入门

: Vue技术栈
发布于 : 2022-10-11

官方文档: https://router.vuejs.org/zh/

项目环境搭建

首先，使用vue-cli新建一个vue项目

vue create blog

然后，进入项目跟路径，安装vue-router

npm install vue-router@4 --save

按照约定，在src目录下创建views和router文件夹，router中创建一个index.js文件，还要在vue项目根目录下创建一个vue.config.js配置文件。

现在，vue项目结构如下

基本用法

假设views目录下有一个要进行路由管理的组件Home.vue，内容如下

<template>
    <div>
        首页
    </div>
</template>

在router/index.js中，添加如下代码

import { createRouter, createWebHistory } from 'vue-router'
import Home from '../views/Home.vue'

// 定义路由规则
const routes = [
    {
        path: '/',
        name: 'home',
        component: Home,
        meta:{
            
        }
    },
]

// 创建router
const router = createRouter({
    history: createWebHistory(process.env.BASE_URL),
    routes
});

// 导出
export default router

然后，再入口main.js中，使用router

import { createApp } from 'vue'
import App from './App.vue'
import router from './router'


const app = createApp(App);
app.use(router)
app.mount('#app')

最后，在App.vue中，添加router-view标签，其内容将由被vue-router管理的组件替代

<template>
    <router-view></router-view>
</template>

现在，启动vue项目

npm run server

使用全局前置守卫例子

一定要记得调用 next() ，否则，表示路由到此就终结了。

router.beforeEach((to, from, next) => {
    isAuthenticated().then(response=>{
        let isAuth=response.data.ok;

        if(to.name !== 'login' && !isAuth){
            // 如果未登录，跳转到登录页面
            next({name:"login"});
        }
    
        else if (to.name === 'login' && isAuth) {
            // 如果已登录，直接跳转到首页
            next({name:"dashboard"});
        }
        else{
            next();
        }
    });
})

import http from '@/static/js/http'

// 检查用户登录状态
async function isAuthenticated(){
    return await http.get("/admin/status");
}

全局后置钩子

router.afterEach((to)=>{
    // 设置页面标题
    document.title = '百度亿下|' + to.meta.title;
})
