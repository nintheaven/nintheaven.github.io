创建项目

使用create-nuxt-app

create-nuxt-app是nuxt.js团队创建的一个脚手架工具。

首先要确保安装了 npx，运行命令

npx create-nuxt-app <project_name>

或使用 yarn

create nuxt-app <project_name>

然后，按照提示选择要使用的工具或库（项目名称、ui框架、测试框架、服务端框架等）

接下来，npm会自动下载相应的模块。

安装完成后，项目也就创建成功了。nuxt项目目录结构如下

手动创建项目

创建项目文件夹

mkdir blog

cd blog

创建package.json文件

{
    "name": "my-app",
    "scripts": {
        "dev": "nuxt"
    }
}

安装nuxt

npm install --save nuxt

创建nuxt目录结构

启动项目

cd <project-name>

yarn dev

构建并启动项目

yarn build

yarn start

默认运行在 3000 端口，访问 http://localhost:3000/

生成静态站点

yarn generate
