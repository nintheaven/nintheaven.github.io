URI中不能有动词，动作应该用HTTP请求方式标识。

API的版本号不应该出现在URI中，因为在RESTFul风格中，URI表示资源唯一名称。推荐的方式是在HTTP请求头中设置版本号，然后在API网关（例如Nginx）中根据版本号将请求路由到提供对应版本资源的服务器中。
