逗号操作符，可以再for循环的初始化部分或步进部分中，使用逗号分割定义多个变量

for(int i = 1, j = 1; i <= 10; i++, j *= 2) {
    System.out.println(i * j);
}

return

return有2个作用：

  • 指定方法的返回值
  • 退出方法

break

break中断当前循环。

continue

continue将控制权移回当前循环的顶部，不会执行continue之后的操作。

goto

Java并不支持goto跳转，但利用break和continue可以实现有限的goto。

break和continue默认只能中断当前的循环。

Java中，在紧靠循环语句的前方，可以声明一个标签，break和continue利用标签跳转到外层循环。

outer:
for(int i = 0; i < 10; i++) {
    inner:
    for(int j = 0; j < 10; j++) {
        if(i > 6){
            break outer;
        }
        if(j % 2 == 1){
            continue inner;
        }
        System.out.printf("i=%d, j=%d%n", i, j);
    }
}
